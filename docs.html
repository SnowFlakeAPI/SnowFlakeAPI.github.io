Introduction

Welcome to the Snowflake API documentation. Snowflake is a secure and reliable key system designed for simplicity.

Base URL

https://api.snowflake.dev/v1

Validate Key

Check if a license key is valid and retrieve its metadata.

GET /validate?key={key}

Response

// success response
{
  "valid": true,
  "key": "SNOW-XXXX-XXXX",
  "expires": "2025-12-31"
}

Parameters

Parameter Type Description
key string The license key to validate
hwid string Optional hardware ID for binding
