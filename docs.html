Introduction

Welcome to the Snowflake API documentation. Snowflake is a secure and reliable key system designed for simplicity.

Base URL

https://api.snowflake.dev/v1

Features

authentication · validation · management

Authentication

All API requests require authentication using your API key. Include your API key in the request header.

Header

Authorization: Bearer YOUR_API_KEY

Getting Your API Key

You can obtain your API key from the dashboard after creating an account. Keep your API key secure and never share it publicly.

Quick Start

Get started with Snowflake API in minutes.

Step 1: Get API Key

Sign up and retrieve your API key from the dashboard.

Step 2: Make Your First Request

// validate a key
GET /v1/validate?key=SNOW-XXXX-XXXX

Step 3: Handle Response

{
  "valid": true,
  "key": "SNOW-XXXX-XXXX",
  "expires": "2025-12-31"
}

Validate Key

Check if a license key is valid and retrieve its metadata.

GET /validate?key={key}

Parameters

ParameterTypeDescription
keystringThe license key to validate
hwidstringOptional hardware ID for binding

Response

{
  "valid": true,
  "key": "SNOW-XXXX-XXXX",
  "expires": "2025-12-31",
  "hwid": "ABC123"
}

Create Key

Generate a new license key.

POST /create

Body Parameters

ParameterTypeDescription
durationintegerKey duration in days
notestringOptional note for the key
hwid_lockbooleanEnable hardware ID binding

Response

{
  "success": true,
  "key": "SNOW-A1B2-C3D4",
  "expires": "2025-12-31"
}

Delete Key

Permanently delete a license key.

DELETE /delete?key={key}

Parameters

ParameterTypeDescription
keystringThe license key to delete

Response

{
  "success": true,
  "message": "Key deleted"
}

Get Key Info

Retrieve detailed information about a license key.

GET /info?key={key}

Parameters

ParameterTypeDescription

Response

  "expires": "2025-12-31",
  "hwid": "ABC123",
  "note": "Premium user",

Examples

Code examples for common use cases.

Python

import requests

url = "https://api.snowflake.dev/v1/validate"
headers = {"Authorization": "Bearer YOUR_API_KEY"}
params = {"key": "SNOW-XXXX-XXXX"}

response = requests.get(url, headers=headers, params=params)
print(response.json())

JavaScript

const response = await fetch(
  "https://api.snowflake.dev/v1/validate?key=SNOW-XXXX-XXXX",
  { headers: { "Authorization": "Bearer YOUR_API_KEY" } }
);

const data = await response.json();
console.log(data);

cURL

curl -X GET "https://api.snowflake.dev/v1/validate?key=SNOW-XXXX-XXXX" \
  -H "Authorization: Bearer YOUR_API_KEY"
